# GymSync PWA Icons

Add the following icon files to this directory:

- `icon-192.png` — 192×192px PNG
- `icon-512.png` — 512×512px PNG

You can generate them from the GymSync logo using any image editor
or a tool like https://realfavicongenerator.net/

Until real icons are added, the PWA will still install but will use
the browser's default icon.
